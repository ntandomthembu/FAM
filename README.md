# FMD Control Platform

The FMD Control Platform is a comprehensive system designed to manage and control outbreaks of Foot-and-Mouth Disease (FMD) in livestock. This platform connects farmers, veterinarians, government authorities, and other stakeholders to ensure rapid detection, containment, and response to disease outbreaks.

## Core Features

1. **Disease Incident Reporting**: A user-friendly interface for farmers to report suspected disease cases quickly and efficiently.

2. **Outbreak Intelligence Map**: A live map displaying suspected and confirmed outbreaks, containment zones, and risk areas, allowing authorities to monitor disease spread in real-time.

3. **Veterinary Investigation System**: A streamlined process for veterinarians to investigate reported cases and update their status in the system.

4. **Livestock Traceability**: Comprehensive tracking of livestock movements, purchases, and interactions to identify potential sources of outbreaks.

5. **Quarantine & Movement Control**: Management of quarantine zones and livestock movement permits to prevent the spread of disease.

6. **Vaccination Campaign Management**: Efficient tracking and management of vaccination campaigns to ensure timely responses during outbreaks.

7. **Early Warning Alerts**: Notifications sent to farmers about nearby outbreaks and movement restrictions to help them take preventive measures.

8. **Farm Biosecurity Guidance**: Resources and guidelines for farmers to enhance biosecurity measures on their farms.

9. **AI Disease Detection (Future Feature)**: Integration of AI technology to analyze uploaded images for early detection of FMD symptoms.

10. **National Disease Dashboard**: A centralized dashboard for authorities to monitor outbreak data, response times, and vaccination coverage.

11. **Export & Trade Compliance**: Generation of necessary documentation to maintain compliance with export regulations during outbreaks.

## Getting Started

To get started with the FMD Control Platform, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd fmd-control-platform
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Set up environment variables by copying the example file:
   ```
   cp .env.example .env
   ```

5. Start the application:
   ```
   npm start
   ```
